# 02_SITE_ARCHITECTURE.md

# English Galaxy Academy

## Site Architecture Overview

The platform must be designed as a scalable educational content system.

All content should be organized according to educational level, content type, curriculum alignment, and user needs.

The architecture must support future growth without requiring major restructuring.

# URL Design Rules

URLs must:

* Be lowercase
* Use hyphens
* Avoid special characters
* Be SEO friendly
* Be human readable

Example:

Correct:

/middle-school/grade-7/unit-3-biographies

Incorrect:

/MiddleSchool/G7/U3

# Architecture Principle

The platform must prioritize:

1. Easy navigation
2. Educational clarity
3. Content discoverability
4. Scalability
5. SEO performance
6. Mobile usability
7. Teacher accessibility
8. Student engagement
