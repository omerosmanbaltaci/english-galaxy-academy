# English Galaxy Academy

## UI/UX Design System

The platform must provide a modern, engaging, educational, and highly accessible user experience for students, teachers, parents, and independent learners.

The design language should communicate trust, clarity, and motivation.

---

# Design Principles

The user interface should be:

* Clean
* Modern
* Educational
* Professional
* Friendly
* Mobile-first
* Accessible

Avoid visual clutter.

Content must always remain the primary focus.
