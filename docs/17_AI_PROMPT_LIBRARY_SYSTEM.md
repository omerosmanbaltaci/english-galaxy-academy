# English Galaxy Academy

## AI Prompt Library System

This document defines the standardized prompt architecture used by all AI content generation tools.

The goal is to ensure consistency, quality, curriculum alignment, and predictable outputs.

---

# Prompt Philosophy

Every prompt must:

Follow educational goals

Support curriculum alignment

Maintain output consistency

Produce reusable resources

Be teacher-friendly

Be scalable
