# English Galaxy Academy

## Admin Dashboard Specification

This document defines the complete administration dashboard structure, modules, workflows, permissions, and user experience requirements.

The dashboard should allow administrators, editors, content managers, and teachers to efficiently manage educational resources, curriculum data, users, AI-generated content, and platform operations.

The dashboard should be:
